# Windows-Media-Player-11-For-Windows-7
A configuration tool aiming to run Windows Media Player 11 on Windows 7
> [!CAUTION]
> 杀毒软件将直接清除提权工具。请提前加入软件解压缩目录到白名单，必要时可能需要完全关闭杀毒软件。

> [!CAUTION]
> 该项目**完全以个人经验完成，请明确了解风险并谨慎使用该项目**

> [!CAUTION]
> **使用本工具需要一定的计算机基础**  
> 本工具涉及系统文件替换、注册表修改等操作，适用于熟悉以下概念的用户：
>
> - 系统还原点
> - UAC（用户账户控制）
>
> 如果你不熟悉上述术语，建议请懂电脑的朋友协助操作。因操作不当导致的系统问题，作者不承担责任。

> [!WARNING]
> 本工具涉及系统文件替换，操作前请务必创建系统还原点。 
> ⚠️ **重要提醒**：操作前建议备份数据。

> [!TIP]
> 运行前建议先关闭所有正在运行的 Windows Media Player 与 Windows Media Center 进程。

## 项目简介
本项目旨在将 Windows Media Player 12 降级为 Windows Media Player 11。
若要使用更旧的版本，请移步至[此处](https://github.com/trustedinstallera-cell/Windows-Media-Player-9-For-Windows-7/releases)
